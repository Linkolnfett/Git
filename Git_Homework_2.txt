TXT
 1. Создать внешний репозиторий c названием TXT.
	(New, Repository name TXT,add a readme file +, create )
 2. Клонировать репозиторий TXT на локальный компьютер.
	git clone https://github.com/Linkolnfett/TXT.git
 3. Внутри локального TXT создать файл “new.txt”.
 	cd txt
	touch new.txt
 4. Добавить файл под гит.
	git add .
 5. Закоммитить файл.
	git commin -m "new"
 6. Отправить файл на внешний GitHub репозиторий.
	git pushvim
 7. Отредактировать содержание файла “new.txt” - написать информацию о себе 
(ФИО, возраст, количество домашних животных, будущая желаемая зарплата).
 Всё написать в формате TXT.
	vim new.txt
	i (button)
 Name: Pavel Skorik, Age:34, Pets:1, Salary requirement:700-900$
	ESC (button)
	:x
 8. Отправить изменения на внешний репозиторий.
	git commit -am "changes"
	git push
 9. Создать файл preferences.txt
	touch preferences.txt
 10. В файл preferences.txt” добавить информацию о своих предпочтениях 
(Любимый фильм, любимый сериал, любимая еда, любимое время года,
 сторона которую хотели бы посетить) в формате TXT.
 	vim preferences.txt
	i (button)
        Favorite movie:Star Wars,Favorite series:Scrubs,Favorite food:Okroshka,
        Favorite season:Summer,Country I'd like to visit:China
	ESC
	:wq
 11. Создать файл sklls.txt добавить информацию о скиллах которые будут 
изучены на курсе в формате TXT
	touch skills.txt
 12. Сделать коммит в одну строку.
 	git add . && git commit -am "2 files"
 13. Отправить сразу 2 файла на внешний репозиторий.
	git push
 14. На веб интерфейсе создать файл bug_report.txt.
	-add file
	-Create new file
	bug_report.txt
 15. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
	Push the button and comments changes
 16. На веб интерфейсе модифицировать файл bug_report.txt, 
добавить баг репорт в формате TXT.
	Click file
	Edit this file (Button)
 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
	Commit changes (Button)
 18. Синхронизировать внешний и локальный репозиторий TXT
	git pull
